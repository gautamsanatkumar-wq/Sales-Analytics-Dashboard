# Sales-Analytics-Dashboard
Sales Analytics Dashboard built using Microsoft Excel to analyze total sales, margins, customer behavior, service performance, and regional trends. The dashboard provides interactive insights with year-wise filtering, sales trends, new vs repeat customer analysis, top customers, and department-wise margins to support data-driven decision-making.
